# pipex-42
A 42 project to learn fork, redirection, and pipe in C programming.

```shell
./pipex file1 cmd1 cmd2 file2 
```
The execution of the pipex program should do the same as the next shell command:
```shell
< file1 cmd1 | cmd2 > file2
```
